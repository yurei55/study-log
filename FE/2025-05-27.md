## 問4. Javaコードの実行結果はどれか？

```java
int[] nums = {1, 2, 3, 4};
int sum = 0;
for (int i = 0; i < nums.length; i++) {
    if (nums[i] % 2 == 0) {
        sum += nums[i];
    }
}
System.out.println(sum);
```
- [ ] A. 4
- [x] B. 6 ✅
- [ ] C. 10
- [ ] D. 2
